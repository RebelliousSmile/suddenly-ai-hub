# 📦 Versions

## Semantic Versioning (SemVer)

Format: `MAJOR.MINOR.PATCH`

- **MAJOR**: Changements incompatibles
- **MINOR**: Nouvelles fonctionnalités (backwards compatible)
- **PATCH**: Corrections de bugs (backwards compatible)
