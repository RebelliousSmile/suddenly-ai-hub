## 🔄 Workflow AIDD

Utiliser le framework AIDD pour la planification :

1. **Create Plan**: `aidd plan create --name "..."`
2. **Decompose**: `aidd decompose --plan plan.md`
3. **Challenge**: `aidd challenge --plan plan.md --review reviews.md`
4. **Execute**: Exécuter les tâches
5. **Review**: `aidd review --plan plan.md`

## 📝 Templates

- **Task Card**: Templates pour les tâches individuelles
- **Epic Card**: Templates pour les épics (groupes de tâches)
